BIENVENIDOS A TINA*S SHOES

El proceso para fabricar calzado no ha variado significativamente a lo largo del tiempo, la elaboración se realiza con máquinas mecánicas y se trata de un proceso artesanal con participación muy reducida de maquinaria ya que la elaboración del producto se realiza básicamente a mano con técnicas rudimentarias. Aunque hay varios tipos de calzado, como son el zapato deportivo, las sandalias, zapatillas, calzado de gamuza u otros; el proceso de fabricación es básicamente el mismo.
